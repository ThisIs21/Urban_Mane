✏️  MODIFIKASI CEPAT
═════════════════════

▶ Ubah Warna Gold:
  Cari: --gold: #C9A84C;
  Ubah jadi: --gold: #D4AF37;

▶ Ubah Font Size Judul:
  Cari: .adm-title { font-size: 26px; }
  Ubah jadi: font-size: 24px;

▶ Ubah Spacing Grid:
  Cari: .stats-grid { gap: 12px; }
  Ubah jadi: gap: 16px;

▶ Ubah Border Radius:
  Cari: border-radius: 12px;
  Ubah jadi: border-radius: 8px;

▶ Ubah Animation Speed:
  Cari: 0.4s ease
  Ubah jadi: 0.3s ease (faster) atau 0.6s ease (slower)


🔍 FIND & SEARCH COMMANDS
═══════════════════════════

Cari semua stat-card:
  grep -n "stat-card" src/pages/admin/AdminDashboardStyles.js

Cari semua panel:
  grep -n "\.panel" src/pages/admin/AdminDashboardStyles.js

Cari semua button:
  grep -n "\.adm-btn" src/pages/admin/AdminTableStyles.js

Cari semua hover:
  grep -n ":hover" src/pages/admin/AdminDashboardStyles.js

Cari semua animasi:
  grep -n "@keyframes" src/pages/admin/AdminDashboardStyles.js

Lihat class names:
  grep -oE '\.[a-z0-9-]+' src/pages/admin/AdminDashboardStyles.js | sort | uniq

Hitung jumlah rules:
  grep -c "{" src/pages/admin/AdminDashboardStyles.js

Lihat ukuran file:
  du -h src/pages/admin/AdminDashboardStyles.js


💾 BACKUP & RESTORE
════════════════════

Backup sebelum edit:
  cp src/pages/admin/AdminDashboardStyles.js src/pages/admin/AdminDashboardStyles.js.backup
  cp src/pages/admin/AdminTableStyles.js src/pages/admin/AdminTableStyles.js.backup

Restore jika salah:
  cp src/pages/admin/AdminDashboardStyles.js.backup src/pages/admin/AdminDashboardStyles.js
  cp src/pages/admin/AdminTableStyles.js.backup src/pages/admin/AdminTableStyles.js

Lihat perbedaan (diff):
  diff src/pages/admin/AdminDashboardStyles.js src/pages/admin/AdminDashboardStyles.js.backup


⚙️  MASS MODIFICATIONS (sed command)
═════════════════════════════════════

Ubah semua #C9A84C jadi #D4AF37:
  sed -i 's/#C9A84C/#D4AF37/g' src/pages/admin/AdminDashboardStyles.js

Ubah semua 12px jadi 8px:
  sed -i 's/12px/8px/g' src/pages/admin/AdminTableStyles.js

Ubah semua 0.4s jadi 0.3s:
  sed -i 's/0\.4s/0.3s/g' src/pages/admin/AdminDashboardStyles.js

Ubah semua font-size: 14px jadi 13px:
  sed -i 's/font-size: 14px/font-size: 13px/g' src/pages/admin/AdminTableStyles.js

💡 TIPS & TRICKS
═════════════════

1. Gunakan F12 (DevTools) untuk inspect element
2. Press Ctrl+Shift+C untuk color picker
3. Press Ctrl+Shift+R untuk hard refresh cache
4. npm run dev → auto-reload saat save
5. Filter DevTools untuk melihat hanya applied styles
6. Gunakan grep untuk find & replace text
7. Backup sebelum mass changes dengan sed
8. Test changes di mobile view (responsiveness)


🆘 TROUBLESHOOTING
════════════════════

CSS tidak apply?
  → Hard refresh: Ctrl+Shift+R
  → Check class names di component
  → Lihat DevTools Console
  → Restart: npm run dev

Animation tidak jalan?
  → Check @keyframes definition
  → Pastikan animation name match
  → Cek timing values
  → Lihat DevTools Performance tab

Warna salah?
  → Verifikasi hex color code
  → Check CSS variables definition
  → Use DevTools color picker
  → Clear cache browser

🏁 WORKFLOW WORKFLOW
══════════════════════

1. Buka CSS file:
   code src/pages/admin/AdminDashboardStyles.js

2. Cari elemen yang ingin diubah:
   Ctrl+F → ketik class name

3. Edit CSS sesuai kebutuhan

4. Save file (Ctrl+S)

5. Browser auto-reload (sudah running npm run dev)

6. Cek hasil di browser

7. Jika OK → selesai
   Jika tidak → undo (Ctrl+Z) dan coba lagi

8. Jika error berat → restore dari backup
